Three rules for creating IMAGES

  1. Save images in the right format(either jpeg, gif or png)
  2. Save images at the right size
    • Same width and height
    • Must be smaller than the width and height
  3. Measure image height and width in pixels(not centimetres or inches)

Advice:
Whenever an image has different colors, you must use image format JPEG. Use GIF or PNG when saving an image with few colors or large areas of the same color (aka flat color).

figure and figcaption tags

  • table tag
  • th tag (table heading)
  • tr tag (table row)
